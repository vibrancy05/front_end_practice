Text Properties

Text properties are used to set the text formatting of an element. Text properties include:

  • font-family
  • font-size
  • font-style
  • font-weight
  • text-align
  • text-decoration
  • text-transform

Color

The color property is used to set the color of the text. The color property can be set using:

  • Color names (e.g. red, blue, green)
  • RGB values (e.g. rgb(255, 0, 0))
  • Hex values (e.g. #ff0000)

Font Family

The font-family property is used to set the font of the text. The font-family property can be set using:

  • Font names (e.g. Arial, Times New Roman, Verdana)
  • Generic font families (e.g. serif, sans-serif, monospace)

Font Size

The font-size property is used to set the size of the text. The font-size property can be set using:

  • Absolute values (e.g. 12px, 16px, 24px)
  • Relative values (e.g. 1em, 1.5em, 2em)

Font Style

The font-style property is used to set the style of the text. The font-style property can be set using:

  • Normal (e.g. normal)
  • Italic (e.g. italic)
  • Oblique (e.g. oblique)

Font Weight

The font-weight property is used to set the weight of the text. The font-weight property can be set using:

  • Normal (e.g. normal)
  • Bold (e.g. bold)
  • Bolder (e.g. bolder)
  • Lighter (e.g. lighter)
